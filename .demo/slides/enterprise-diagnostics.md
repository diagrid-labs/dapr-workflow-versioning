# Enterprise Diagnostics

Workflow that performs diagnostics for the starship Enterprise.

The workflow will analyze the following:

- Hull
- Warp core
- Security systems
- (Weapon systems)

The workflow combines the results from all these analyses and provides recommendations on what requires attention and sends a notification to the bridge.
